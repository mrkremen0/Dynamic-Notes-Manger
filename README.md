# TaskFlow — Интерактивный менеджер заметок
Простое и быстрое приложение для управления задачами с сохранением данных.

### Технологии:
- HTML5 / CSS3 (Flexbox) — адаптивная верстка.
- JavaScript (Vanilla) — логика приложения без библиотек.
- LocalStorage — сохранение ваших заметок после перезагрузки страницы.

### Функционал:
- Создание и удаление заметок с уникальными ID (Date.now()).
- Проверка на пустой ввод (валидация).
- Кастомный дизайн скроллбара и адаптация под мобильные устройства.
